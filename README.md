# Reto de 30 Días: Respiración Terapéutica y Bioenergética

Este repositorio contiene la landing page premium para el **Reto de 30 Días de Respiración Terapéutica y Bioenergética** de **Alquimia Somática**.

Se trata de una página de venta de un solo archivo (Single-File Landing Page) de altísimo nivel visual y alta conversión, diseñada para ser cargada de manera instantánea y ofrecer una experiencia interactiva inmersiva.

---

## Características Principales

- **Diseño Orgánico y Minimalista (Warm Somatic):** Una cuidada paleta en tonos musgo, salvia, arena y oro viejo sobre fondo ultra oscuro, acompañada de una textura táctil analógica (grano SVG).
- **Fondo de Partículas Dinámico (Canvas):** Animación nativa de partículas flotantes que modulan su ritmo de escala de forma colectiva, simulando una inhalación y exhalación diafragmática profunda.
- **Simulador de Respiración Somática:** Un orbe completamente interactivo que permite al visitante experimentar tres patrones de respiración reales y sentir sus efectos en 1 minuto:
  - *Coherencia Cardíaca* (Equilibrio rápido del pulso)
  - *Bio-Carga Energética* (Activación profunda de volumen de oxígeno y descarga de tensión)
  - *Calma Profunda 4-7-8* (Desactivación del estado de alarma y rumiación)
- **Estructura de Copywriting Persuasivo:** Explicación humana paso a paso del bucle de feedback por WhatsApp (sesión diaria, feedback por audio o texto, rediseño diario personalizado), explicación científica de la *coraza muscular*, fases del reto, testimonios reales, oferta con cupos limitados y acordeones de FAQ dinámicos.
- **Micro-interacciones y Animaciones Fluidas:** Implementación de revelaciones sutiles al hacer scroll utilizando la biblioteca ligera **Anime.js** y animaciones fluidas del orbe y anillos concéntricos.

---

## Tecnologías Utilizadas

- **HTML5 Semántico**
- **Tailwind CSS v4 CDN** (configurado con variables de tema nativas `@theme`)
- **Anime.js v3.2.2** (para las animaciones de scroll e interactividad del orbe)
- **Lucide Icons** (iconos minimalistas limpios)
- **HTML5 Canvas API** (para las partículas respiratorias)

---

## Cómo Ejecutar

Al tratarse de una landing page optimizada sin dependencias de compilación ni frameworks pesados, puedes ejecutarla directamente:

1. Clona el repositorio:
   ```bash
   git clone https://github.com/pablozamit/respiracionterapeutica.git
   ```
2. Abre el archivo `index.html` en cualquier navegador web moderno (Chrome, Safari, Firefox, Edge).
